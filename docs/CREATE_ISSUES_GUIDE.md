# How to Create GitHub Issues from Templates

This guide explains how to create GitHub issues for the planned RugWatch features using the issue templates.

## Quick Start

The issue templates are now available in the repository. You have two main options for creating issues:

### Option 1: Using GitHub Web Interface (Recommended)

1. Go to: https://github.com/GrooveTechie/rugpull-scam-token-detection/issues/new/choose
2. Select the feature template you want
3. Review the pre-populated content
4. Submit the issue

### Option 2: Manually Copy Templates

1. Browse templates in `.github/ISSUE_TEMPLATE/`
2. Copy the content
3. Create a new issue and paste
4. Add appropriate labels

## Available Templates

**Tier 1:** Authority Mutation Watcher, Dev Wallet Graph, Liquidity Event Classifier
**Tier 2:** Early Holder Persistence, Sell Concentration Index
**Tier 3:** Funding Source Clustering, Communication Correlates
